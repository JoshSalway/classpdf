About ClassPDF

ClassPDF is a set of free PDF tools built for teachers. Merge, split, compress and convert PDFs right in your browser - no sign-up, no file limits, no watermarks. Because everything runs on your own computer, your files (and your students' work) never leave your device.

Why we built it

Teachers shouldn't have to upload a class set of worksheets to a random website just to combine them into one PDF or shrink a file so it fits the school LMS. So we made simple, private tools that do the job in seconds and cost nothing.

Who made it

ClassPDF is made by the team behind SproutSheets, free printable worksheets for teachers. It's open source and built on BentoPDF under AGPL-3.0 - see the source code on GitHub for the source code.
